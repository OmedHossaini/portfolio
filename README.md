# Omed Hossaini — Portfolio

Personal one-page developer portfolio — **React + Vite**, plain CSS, Space
Grotesk. Static, no backend, deployed on Vercel.

The app lives in [`frontend/`](frontend). See
[frontend/README.md](frontend/README.md) for content editing, image slots, and
full deploy steps.

## Quick start

```bash
cd frontend
npm install
npm run dev
```

## Deploy (Vercel)

- Import this repo → **Root Directory:** `frontend` → **Node.js:** `24.x`
- Vite is auto-detected; no environment variables needed.
- 🔗 Live: _add your Vercel URL here once deployed_
